Let's talk apithreads

July 2023 | npm Package | Reverse engineering | Fork on GitHub

apithreads is an unofficial, Reverse-Engineered Node.js/TypeScript client for Meta's Threads.

Features

— Post Thread with text/image/link attachment
— Reply to threads
— Like/Unlike threads
— Follow/Unfollow User

Try it Out

yarn add apithreads
npm install apithreads
pnpm add apithreads
