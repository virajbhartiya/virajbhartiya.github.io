Let's talk Alien

 June 2023 | Virtual Screen Sharing | Fork on GitHub

Alien is an innovative desktop application that transforms any device with a web browser into a convenient secondary screen for your computer, utilizing a wireless connection. With Alien, you can effortlessly mirror your entire computer display onto any device screen equipped with a web browser. Additionally, it offers the flexibility to select and share a specific application window, making it perfect for presentations and focused content sharing.

The standout feature of Alien lies in its ability to utilize any device as a secondary screen. For a truly immersive extended desktop experience, Alien seamlessly integrates with Virtual Display Adapter. This combination enhances productivity and multitasking capabilities by expanding your workspace across multiple screens.

Notably, Alien goes beyond traditional screen mirroring functionalities. You can leverage the application to transform your tablet or smartphone into a teleprompter without the need for a Virtual Display Adapter. By enabling the Flip Screen Mode, Alien mirrors your computer screen horizontally within the web browser of your tablet, creating a convenient teleprompter setup.

In summary, Alien revolutionizes the way you utilize screens by offering a versatile solution that transforms any device with a web browser into a secondary screen. Whether you need to mirror your entire display or focus on specific application windows, Alien provides seamless functionality. Experience true productivity and creativity by using Alien with Virtual Display Adapter, or harness the convenience of a teleprompter with Flip Screen Mode on your tablet or smartphone.

System Requirments

— Operating System: Windows | Linux | MacOS
— Available Disk Space: 210MB
— RAM: 250MB on average to run app with one screen sharing session. Every new screen sharing session will need upto 100MB of extra memory on average for smooth performance.

Development

Setting Up

Note: Built using Node version 14.15.0

git clone https://github.com/virajbharitya/Alien
cd Alien
cd app/client && yarn install --frozen-lockfile && cd ../../ && yarn install --frozen-lockfile
yarn dev

Scripts

yarn start — run in production mode to test, without packaging
yarn test — Run all unit tests
yarn package-[win | linux | mac] — to package the app and make executables available in release folder
View the package.json file for all the commands
